Для запуска сборки нужно склонировать репозиторий:
```
$ git clone git@github.com:**************.git
```
перейти в склонированный репозиторий и выполнить
```
$ git checkout new-project
```
перейти в папку с проектом
```
$ cd new-project
```
установить зависимости:
```
$ npm install
```
запустить сборку:
```
$ npm run dev
```
